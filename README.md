# 📝 Modern Interactive To-Do List App

A sleek, responsive, and feature-rich task management application built with vanilla web technologies.

## ✨ Features

* **Priority Filtering:** Organize your tasks by High, Medium, or Low priority to stay focused on what matters most.
* **Bilingual Support:** Easily switch between **English** and **Sinhala** languages.
* **Dark Mode:** Aesthetic Light and Dark themes to protect your eyes and suit your style.
* **Interactive UI Sounds:** Premium "Tink" and "Pop" sounds for a satisfying user experience when adding or deleting tasks.
* **Persistent Storage:** Uses **Local Storage** so your tasks remain saved even if you refresh the page.
* **Recycle Bin:** Restore accidentally deleted tasks from the "Deleted Tasks" section.
* **Search Functionality:** Quickly find specific tasks using the built-in search bar.


## 🛠️ Built With

* **HTML5** - Semantic structure
* **CSS3** - Custom styling, Flexbox, and Theme variables
* **JavaScript (ES6)** - Dynamic DOM manipulation and Logic
* **FontAwesome** - Modern UI icons

## 📂 Installation & Usage

1.  Clone the repository:
    ```bash
    git clone [https://github.com/SewminiDilshani/To-Do-List.git](https://github.com/SewminiDilshani/To-Do-List.git)
    ```
2.  Open `index.html` in your favorite browser.
